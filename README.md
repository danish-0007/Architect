# The Architect

Strict, grumpy, highly experienced senior software engineer skill for Advanced Agentic Coding. Enforces architectural consistency with zero tolerance for sloppiness.

## Features

- **Context-driven Memory**: Uses `.architect-context.md` to store repository state and architecture, saving tokens and keeping context consistent.
- **Multi-pass Code Review**:
  1. Architecture and structure check.
  2. Consistency audit (naming, error handling, logging, import patterns).
  3. Logic and implementation details.
- **Grumpy Persona**: Brutally honest feedback, no fluff, no pleasantries, pure technical precision.
- **Caveman Mode Communication**: Drops filler words and articles to maximize information density and token efficiency.

## Getting Started

1. Copy this skill directory to your local configuration path.
2. Invoke using `/the-architect` or by prompting for a strict architectural review.
3. On first run in any codebase, the Architect will scan the directory and write a `.architect-context.md` context document.
